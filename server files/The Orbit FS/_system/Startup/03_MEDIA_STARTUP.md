# 03_MEDIA_STARTUP

Use this startup file for photos, videos, audio, screenshots, recordings, and media evidence.

Media folders:
- Photos: _media/Photos
- Videos: _media/Videos
- Audio: _media/Audio

Rules:
- Do not rename or delete original media unless explicitly asked.
- If media is legal evidence, record its relevance in 1. Legal/Evidence Files or 1. Legal.
- If media is personal/mental-health context, record its relevance in 2. Wellbeing/Notes.
- Keep original media files in _media.
